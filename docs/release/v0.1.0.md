## Release Notes

`tokenmaxxing-ai` installs portable approval-first workflows for AI coding agents:

- `/simplify` finds over-engineered code, presents a plan, asks approval, edits, and verifies.
- `/review lite|mid|ultra` reviews repositories with structured findings, confidence levels, fix plans, approval gates, and verification.
- `.tokenmaxxing.md` lets repeated runs keep concise project-local memory.

Supported targets:

- Codex
- Claude Code
- Cursor
- GitHub Copilot
- OpenCode
- CommandCode
- Antigravity
- Kiro
